# Epic E02: Expert Has a Managed Profile and Can Be Matched

## Purpose

An AI integration expert can create a profile with their specialisations, target client criteria, and availability — so they can be matched to relevant prospects automatically and appear credibly to prospects browsing results.

> **Layer constraint:** This epic delivers Layer 3.1 UI only (callibrate.io). Auth, profile APIs, and Cal.com integration are Layer 2 — already built in E06S03 and E06S04. E02 consumes those APIs and renders the expert-facing UX.

---

## Scope

- Expert onboarding and account creation (Supabase Auth)
- Profile builder: specialisations, verticals, budget range, languages, bio, geo zone, work mode
- **Market positioning profiling** (E02S02): expert describes their positioning → system generates `expert.preferences` JSONB. Profiling dimensions: career stage (junior / medior / senior / high-ticket), work mode (remote / on-site / hybrid), availability (full-time freelance / side projects / X days/week), budget range, project stage accepted (exploration / defined scope / urgent execution), languages, geo zone, industries. Expert does not configure raw matching criteria — they describe their positioning.
- Availability integration with Cal.com (managed user configured in E06S04)
- Expert dashboard: incoming leads, match scores, booking status

---

## Out of Scope

- Expert billing and subscription (E05)
- AI-generated profile content
- Multi-language profile UI (MVP is French)

---

## Stories

- E02S01: Expert can create an account and set up their profile
- E02S02: Expert completes market positioning profiling → generates expert.preferences JSONB (career stage, work mode, availability, budget range, project stage, languages, geo zone, industries)
- E02S03: Expert can complete Cal.com Google Calendar connection from their onboarding UI
- E02S04: Expert can view their incoming matched leads in a dashboard

---

## Success Metrics

- 10 expert profiles active with all criteria filled
- Expert rating of profile setup experience ≥7/10
- All expert profiles visible in matching results

---

## Dependencies

- E01 (Offer Validated) — experts recruited via Hand Raiser campaign
- E06 (Platform Infrastructure) — auth, profile API, Cal.com integration must exist
