## Key Design Decisions

- **DEC-34 applied**: `cal.setup` is synchronous in `POST /api/experts/register`. If Cal.com API is unavailable, failure is logged non-fatally (AC9).
- **DEC-33 honoured**: No `matching-jobs` queue consumer ‚Äî AC1 reinterpreted as synchronous integration.
- **Match linking via metadata**: Webhook handler reads `payload.metadata.match_id` (set by E06S07 booking URL construction) to link bookings to matches. Falls back to `organizer.username` ‚Üí `cal_username` lookup for expert_id.
- **HMAC**: Uses Web Crypto API (`crypto.subtle`) ‚Äî available in Cloudflare Workers runtime, no external deps.
