## Issues Found

### Minor — Dev env TURNSTILE_SECRET_KEY not documented

**Severity:** Low (documentation gap, not a runtime defect)

The top-level wrangler.toml (used by `wrangler dev`) has no `[vars]` section and `.dev.vars.example` does not include `TURNSTILE_SECRET_KEY`. A developer setting up locally would need to manually add `TURNSTILE_SECRET_KEY=1x0000000000000000000000000000000AA` to their `.dev.vars` file. The staging env is correctly configured with the test key in `wrangler.toml`, so CI and deployed staging always pass.

**Recommendation:** Add `TURNSTILE_SECRET_KEY=1x0000000000000000000000000000000AA` to `.dev.vars.example` (test key is safe to document — it is publicly listed in Cloudflare's own documentation).
