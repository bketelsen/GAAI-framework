## Edge Cases

| AC | Edge Case | Resolution |
|----|-----------|------------|
| AC1 | Hint always visible | No JS toggles `#freetext-hint`. Static `<p>` always in DOM. |
| AC2 | `config.locale = undefined` (old satellites) | `config.locale != null` guard ŌåÆ English default. |
| AC2 | `config.locale = "fr-BE"` | `startsWith('fr')` ŌåÆ French. |
| AC2 | Unsupported locale | Falls to `else` ŌåÆ English default. |
| AC4 | Empty textarea | `len > 0` condition prevents spurious warning. |
| AC4 | Warning during loading state | `clearTimeout + hideSoftWarning()` in submit handler. |
| AC5 | Multiple ids in aria-describedby | Space-separated list per HTML spec. Hint first (most relevant on fresh focus). |
