## Key Decisions

### AC2 — HMAC polyfill
`crypto.subtle.timingSafeEqual` not available in CF Workers as of 2026-02.
Polyfill: `generateKey(HMAC/SHA-256) as CryptoKey → sign × 2 → byte-compare`.
`as CryptoKey` cast required because `generateKey` TS return type is `CryptoKey | CryptoKeyPair`.

### AC3/AC4/AC5 — Zod v4
Zod v4 (^4.3.6 in package.json): use `.issues` not `.errors` on ZodError.
`z.string().email()`, `z.string().min(1).max(255)`, `z.string().datetime()`.
Temporal checks via `.superRefine()` for start < end and start > now.

Already a declared dependency (`zod: ^4.3.6`). No install needed.
