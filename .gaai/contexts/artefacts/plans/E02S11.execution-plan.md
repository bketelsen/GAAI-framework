## Key Technical Decisions

- **Hybrid model (DEC-118):** DB rules → JSONB fallback → `not_configured`. GCal is Layer 2, now optional (non-blocking).
- **Timezone:** `experts.timezone TEXT NOT NULL DEFAULT 'UTC'`. `expandRulesToWorkingHours()` converts local TIME to UTC using Intl offset computation.
- **Soft delete only:** `is_active = false`. RLS: SELECT + INSERT + UPDATE (no DELETE).
- **Overlap detection:** Application-level: `s1 < e2 && e1 > s2`. Excludes self on PUT.
- **Dashboard:** Mon-first European display (day_of_week 0=Sun → last column). 06:00–22:00, 32 rows × 30 min.
