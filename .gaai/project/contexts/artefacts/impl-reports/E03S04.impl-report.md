## Files Created

### `workers/frontend/satellites/src/pages/experts-directory.ts`

- Async `renderExpertsDirectoryPage(config, posthogApiKey, coreApiUrl): Promise<string>`
- Fetches expert listing from Core API (`/api/experts/public?vertical=...&per_page=12&page=1`)
- Renders: branded header, headline with count, skill filter bar, 3-col card grid, load-more button
- Each card: avatar initial (color by tier), tier badge, composite score, headline, top-3 skills, rate range, "Voir le profil" link
- Empty state: "Aucun expert ne correspond à ces critères" + reset + /match CTA (AC9)
- JSON-LD ItemList structured data (AC7)
- PostHog: `satellite.directory_viewed`, `satellite.directory_filter_applied` (AC8)
- Client-side JS: `filterCards(skill)` toggle + "Charger plus" pagination via fetch (AC6)
- SEO: `<meta name="robots" content="index, follow">`, canonical `/experts`, dynamic title (AC7)

### `workers/frontend/satellites/src/pages/experts-profile.ts`

- Async `renderExpertProfilePage(config, posthogApiKey, coreApiUrl, slug): Promise<{ html: string; status: number }>`
- Returns `{ status: 404 }` when Core API returns 404, `{ status: 500 }` on network error
- Renders full profile: avatar (initial + tier color), tier badge, composite score, headline
- Availability: "Disponible" + `avail--available` / "Disponible prochainement" + `avail--soon`
- Bio excerpt, skills tags, industries tags, rate range, completed projects, languages
- Primary CTA: `id="cta-match-btn"` `href="/match"` "Vérifier la compatibilité" (AC4)
- Secondary CTA: `id="unlock-trigger-btn"` / inline form `id="unlock-form"` → `create-from-directory` → fires `booking-open` event (AC4)
- AC11 direct CTA: conditional `expert.direct_booking_url ?? null` — renders "Prendre un rendez-vous direct" link with `class="cta-direct"` only when URL is non-null
- JSON-LD Person: XSS-safe (angle brackets escaped in JSON context), anonymized (no real name) (AC7)
- PostHog: `satellite.expert_profile_viewed`, `satellite.expert_cta_match_clicked`, `satellite.expert_cta_unlock_clicked` (AC8)
- Back link "Retour au répertoire" → `/experts`
- SEO: index+follow, canonical `/experts/{slug}`, title with tier+skill+brand (AC7)
