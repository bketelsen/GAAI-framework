## Rule Compliance

- Loader uses `requireSession` — session guard enforced.

- Both API calls wrapped in `.catch()` with typed defaults — graceful degradation per project pattern.

- Helper functions (`getTierConfig`, `formatMonthLabel`, `safePct`) exported for testability — consistent with existing test patterns in this codebase.
