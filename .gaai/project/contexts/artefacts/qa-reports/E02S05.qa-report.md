### Minor Observations (non-blocking)

1. **AC8 badge text:** The badge reads `{windowInfo.daysLeft}j restants` (e.g. "5j restants"). The AC specifies `{N} jours restants`. The abbreviated form is consistent with the French dashboard style and functionally correct — not a defect.
2. **AC9 evaluate toast message:** The toast fires "Évaluation enregistrée — merci, vos futurs matchs s'amélioreront." The AC specifies "toast with improvement message" without mandating exact wording — the message fulfills intent.
3. **Flag button hidden vs disabled when expired:** AC8 says "flag button disabled/hidden". Implementation hides the button (by not rendering it when `windowInfo === null`). This is a valid and arguably better UX choice than disabling — no defect.
