### Known Limitations (documented in impl-report, accepted scope)

1. **Basic email regex**: `[^\s@]+@[^\s@]+\.[^\s@]+` passes some technically invalid emails. Intentional — server-side 422 is the authoritative guard.
2. **`fetchProfilesDirect()` silent error swallow**: 409 branch network failure produces empty `revealFullProfiles([])` call. Acceptable for current scope; flagged for future hardening.
3. **No TypeScript type for `window.__SAT__`**: Consistent with `confirm.ts` and `match.ts` pattern — `window.__SAT__` is set via inline script and used within IIFE string. Not a new risk.
4. **`booking-open` CustomEvent is a stub**: Nothing listens until E03S06 (booking modal). Expected.
