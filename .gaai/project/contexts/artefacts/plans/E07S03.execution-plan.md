### Step 4 — Create `src/lib/posthog.ts`

New file. Core utility:

```typescript
import { PostHog } from 'posthog-node';

const POSTHOG_HOST = 'https://ph.callibrate.io';

export interface CapturePayload {
  distinctId: string;
  event: string;
  properties?: Record<string, unknown>;
}

export function captureEvent(
  apiKey: string | undefined,
  payload: CapturePayload,
): Promise<void> {
  if (!apiKey) return Promise.resolve();

  const ph = new PostHog(apiKey, {
    host: POSTHOG_HOST,
    flushAt: 1,
    flushInterval: 0,
  });

  return ph
    .captureImmediate({
      distinctId: payload.distinctId,
      event: payload.event,
      properties: payload.properties ?? {},
    })
    .catch(() => {
      // AC9: swallow all PostHog errors — never surface to API caller
    });
}
```

**Design:** Per-request PostHog instance (stateless CF Workers). `flushAt: 1` + `flushInterval: 0` = immediate flush. `.catch(() => {})` = AC9 fire-and-forget.

| AC1 | `posthog-node` `workerd` vs `node` export conditions | Mock in all test files |

| AC5 | `duration_min` not in current confirm.ts select | Add to select string |

| AC7 | Survey routes not on staging branch | Merge E06S17 as prerequisite |
