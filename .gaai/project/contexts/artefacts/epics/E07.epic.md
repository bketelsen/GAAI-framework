# Epic E07: Product Analytics & Conversion Optimization

## Purpose

Both conversion funnels (prospect satellite journey and expert platform activation) are observable, measurable, and optimizable — so that the founder and GAAI agents can identify drop-off points, validate UX hypotheses, and make data-driven decisions to increase conversion rates and retention.

---

## Scope

- PostHog EU Cloud integration (GDPR-compliant, cookieless mode)
- Cloudflare Worker reverse proxy for PostHog (ad-blocker bypass, +10-30% event capture)
- Satellite client-side instrumentation: autocapture, session replay, page views, CTA clicks
- Core API server-side event capture: all prospect + expert + booking endpoints
- PostHog funnel dashboards: prospect conversion funnel, expert activation funnel, business metrics
- GAAI analytics-query skill: programmatic PostHog Query API access for agent-driven growth analysis

---

## Out of Scope

- Feature flags & A/B testing infrastructure (phase 2, separate stories)
- PostHog Workflows automation — onboarding nudges, churn intervention emails (phase 2)
- Custom analytics UI/dashboard (PostHog built-in dashboards are sufficient)
- Revenue analytics / Stripe integration (E05 domain)
- Marketing attribution beyond UTM (phase 2)
- Self-hosted PostHog (Cloud EU is sufficient at scale)
- LLM Analytics deep integration (phase 2 — basic event in E07S03)

---

## Stories

- E07S01: PostHog reverse proxy CF Worker
- E07S02: Satellite client-side instrumentation (prospect UX tracking)
- E07S03: Core API server-side event capture (prospect + expert funnels)
- E07S04: PostHog funnel dashboards via API
- E07S05: PostHog MCP integration + GAAI analytics-query skill (agent data access)

---

## Success Metrics

- Full prospect funnel visible: satellite.page_view → cta_clicked → form_submitted → matches_viewed → identified → availability_checked → booking.held → booking.confirmed
- Full expert funnel visible: registered → profile_updated → gcal_connected → lead_received → booking.confirmed
- Drop-off rates quantified per step within 7 days of deployment
- GAAI agent can query funnels and trends programmatically via analytics-query skill
- Session replay available for satellite UX debugging
