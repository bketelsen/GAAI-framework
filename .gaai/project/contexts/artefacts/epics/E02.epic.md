# Epic E02: Expert Dashboard (app.callibrate.io)

## Purpose

An AI integration expert can sign up, complete onboarding, manage their profile and calendar, view incoming leads, track bookings, manage billing, and monitor performance analytics — all through a dedicated dashboard at app.callibrate.io — so they have full visibility and control over their platform experience.

> **Layer constraint:** This epic delivers Layer 3.1 UI only (app.callibrate.io). All backend APIs (auth, profile, matching, booking, billing, scoring) are Layer 2 — already built in E06. E02 consumes those APIs and renders the expert-facing UX.
>
> **Consolidation (DEC-114):** E02 absorbs Layer 3.1 portions of E04 (expert lead page) and E05 (billing dashboard, lead pipeline, analytics). E04 retains only satellite-side booking UX (Layer 3.2). E05 Layer 2 stories already delivered (E06S31–S34).
>
> **Tech stack (DEC-113):** React Router v7 Framework Mode (SSR) on CF Workers via Cloudflare Vite plugin. Server-side loaders eliminate data waterfalls and handle auth redirects before HTML is sent.

---

## Scope

- Expert signup and login (Supabase Auth PKCE flow)
- Multi-step onboarding wizard (profile + expertise + preferences + GCal)
- Profile management (all fields, admissibility criteria, outcome tags)
- Google Calendar connection management
- Lead pipeline with status actions (confirm, flag, evaluate)
- Bookings page (upcoming/past, cancel/reschedule)
- Billing & subscription (credit balance, LS activation, transaction history, spending controls)
- Analytics dashboard (composite score, score breakdown, monthly metrics, conversion)
- callibrate.io expert-facing landing page (separate Worker)

---

## Out of Scope

- Prospect-facing UIs (E03 — Layer 3.2)
- Satellite-side booking widget (E04 — Layer 3.2)
- AI-generated profile content
- Multi-language UI (MVP is French)
- Mobile native app
- Real-time chat between expert and prospect

- Expert signup → profile complete conversion ≥ 70%
