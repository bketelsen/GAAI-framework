# Epic E03: Prospect Gets Qualified and Matched to the Right Expert

## Purpose

A company with a real AI automation need can describe their project, see AI-extracted requirements confirmed, and receive a ranked list of matched experts with transparent per-criterion scores — so they can make an informed choice without wading through irrelevant profiles.

> **Layer constraint:** This epic delivers Layer 3.2 UI only (satellite sites). All extraction, scoring, and matching logic is Layer 2 — already built in E06S07, E06S08, E06S05. E03 consumes those APIs and renders the results. *(callibrate.ai supprimé — DEC-39)*

---

## Scope

- Freetext project description input (calls `POST /api/extract` from E06S08/E06S12)
- Confirmation step UI: displays low-confidence fields as targeted questions (max 2–3), skipped if all fields high-confidence
- Anonymized match results display: top-3 experts visually prominent (DEC-25), global score + per-criterion breakdown (quality_tier badge)
- Email gate: prospect provides email to unlock full expert profiles (protects against disintermediation, aligns with credit billing model)
- Directory mode: browse anonymized expert profiles, filter by vertical/skills, ranked by composite_score
- Inline booking widget: real-time availability, slot selection, hold, confirm (uses E06S11 booking engine — Cal.com superseded by DEC-41)

---

## Out of Scope

- AI extraction logic (Layer 2 — E06S08/E06S12)
- Matching computation (Layer 2 — E06S05, E06S07, E06S22)
- Booking engine APIs (Layer 2 — E06S11)
- Prospect account creation (anonymous at MVP)
- Prospect payment (prospects use the platform free)
- Saved prospect searches or history

## Success Metrics

- Freetext submission rate ≥60% (started → submitted)
- Confirmation step completion rate ≥80% (of those who reach it)
- Email gate conversion rate ≥40% (anonymized views → identify)
- Booking rate ≥15% (profiles unlocked → booking confirmed)
- Average time submission → matched list displayed <5 seconds
- All tracked via PostHog funnel dashboards (E07S04)
