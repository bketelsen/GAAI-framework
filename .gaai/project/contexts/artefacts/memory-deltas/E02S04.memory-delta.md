## New Patterns Worth Recording

### sessionStorage flash pattern for cross-redirect state
First use in dashboard. When a server action triggers a full-page redirect (OAuth flow), state can't be passed through normal React Router mechanisms. Pattern: set `sessionStorage.setItem('flag', '1')` synchronously before form submit → on next page mount, check flag + condition → fire side effect (toast) + `sessionStorage.removeItem('flag')`.

**When to use:** Any cross-redirect notification where the source route (pre-redirect) wants to signal the destination route (post-redirect, different URL).

### apiDelete pattern
`api.server.ts` now supports DELETE method via `apiDelete<T>(env, token, path)`. Added by E02S04. Pattern is identical to `apiGet/apiPost/apiPatch` — consistent.
