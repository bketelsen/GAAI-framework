## Bugs Found and Fixed

1. **KV state pollution across tests** (extract.test.ts): Shared module-level `mockEnv` caused rate-limit counters to accumulate across tests. 13 tests returned false-429. Fixed by resetting `mockEnv = makeMockEnv()` in `beforeEach`.

2. **Flow token missing in prospect test stubs** (prospects.test.ts): AC11 added flow token check before Turnstile. 6 existing prospect tests didn't include `flow_token` → 403. Fixed by adding `signFlowToken` + including token in all affected test requests.

3. **`sql.end is not a function` — CI failures** (post-merge): Stash changes included in E06S40 added `sql.end()` in `finally` blocks to several source files. 20 existing tests failed in CI. Fixed in two commits:
   - `8a6df7a`: `expertPool.test.ts` (6) + `lsBillingCron.test.ts` (8)
   - `66a7337`: `syncExpertPool.test.ts` (6)

4. **Lorem ipsum regex false positive** (inputValidation.test.ts): Test text used "please" repeated many times, inadvertently triggering the `(.{3,})\1` repetition pattern. Fixed by using clean 11-word sentence with no repetition.

5. **Stash bug fix included in PR** (`matches.ts`): Match status was inserted as `'active'` instead of `'pending'`, mismatching the DB schema enum. Fixed as part of this delivery.
