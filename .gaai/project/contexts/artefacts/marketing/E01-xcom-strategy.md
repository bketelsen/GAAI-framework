## 2. Stratégie Cold Start (Zéro Followers)

Le problème d'un compte neuf : portée organique quasi nulle.

**Séquence recommandée :**

| Étape | Action | Délai |
|---|---|---|
| J1 | Créer le compte + compléter la bio | Jour 1 |
| J1-J3 | Follow 50-100 comptes actifs dans le niche (n8n, Make, AI automation, solopreneurs) | J1–J3 |
| J3-J5 | Commenter 5-10 posts populaires de cette audience (vraies réponses, pas du spam) | J3–J5 |
| J5 | Premier post Hand Raiser + $20 promoted tweet | J5 |
| J7 | Analyse des résultats + décision suite | J7 |

> L'objectif des follows + commentaires : apparaître dans le feed de l'audience cible avant de poster, sans follower count.

## 3. Hand Raiser Posts — Expert Side

> Objectif : ≥10 experts qualifiés lèvent la main. Gate E01S02.
> Format X : court + CTA "reply". Pas de thread — un seul post suffit.

### Version A — Pain direct
```
AI automation consultants: how much of your week goes into chasing
unqualified prospects?

If I could send you 2–3 pre-screened leads per month — right budget,
real project, decision-maker ready — would that be worth paying for?

Reply "yes" if that's a real problem 👇
```

### Version B — Feast or famine angle
```
The feast-or-famine cycle in AI automation consulting is real.

Great at the work. Terrible at keeping a steady pipeline.

If there was a way to get qualified inbound leads without LinkedIn
hustle or cold outreach — would you use it?

Reply "me" 👇
```

### Version C — Vocabulaire testé (après résultats Reddit)
```
"Half my week finding decent leads."

If you're doing AI automation work and that line hits — I'm building
something for you.

Reply "me" to hear more when it's ready 👇
```

> Version C n'utilise que du vocabulaire validé organiquement (r/aisolobusinesses).

## 5. Posts d'Observation (No CTA)

> Même logique que Reddit : faire parler l'audience dans ses propres mots.
> Format X : question courte, conversationnelle. Pas de liste à puces — ça sonne corporate.

### Observation A — H12 + vocabulaire
```
Quick question for people doing AI automation work for clients:

Is there a platform where serious buyers find serious specialists?
Or is everyone still just figuring it out on Upwork and LinkedIn?

Genuinely curious what's working in 2026.
```

### Observation B — H2 (temps perdu)
```
AI automation freelancers: honest question.

What % of your week goes into finding/chasing clients vs.
actually doing the work?

Just curious what the real split looks like.
```

### Observation C — H13 (croissance du marché)
```
The number of people calling themselves "AI automation specialists"
has exploded in the last 12 months.

Has actual client demand kept up — or is the market getting crowded
faster than the work is growing?
```

### Observation D — Vocabulaire (pour valider les termes)
```
What do you call yourself when a non-technical client asks what you do?

- "AI automation consultant"
- "workflow architect"
- "AI integrator"
- Something else entirely?

Curious what actually lands.
```

## 8. Critères de Qualification (rappel)

**Expert qualifié :**
- Profil indique une activité réelle en AI automation / intégration / workflow
- Pas un débutant cherchant à apprendre — quelqu'un qui facture déjà des clients

**Prospect qualifié :**
- Profil indique un poste décisionnaire (founder, CEO, ops, directeur)
- Travaille dans une entreprise (pas freelance cherchant du travail)
- Le profil suggère un besoin réel en automatisation
